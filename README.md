# 4주차 날씨 API 실습

응용프로그램 실습 4주차 — Open-Meteo API와 기상청 API를 이용한
날씨 데이터 수집 및 엑셀 저장 실습 코드

## 파일 설명

| 파일 | 설명 |
|------|------|
| weather.py     | Open-Meteo API 날씨 조회 (API 키 불필요) |
| kma_weather.py | 기상청 초단기실황 API 날씨 조회 |

## 실행 방법

pip install requests
python weather.py
